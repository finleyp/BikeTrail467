# Bike Trails

![alt text](https://github.com/finleyp/BikeTrail467/wiki/media/bike-icon.png "Bike Trail Logo")

A cross platform mobile application built using Flutter to allow users the ability to record bike trails.

Features:

* Trail Recording
* Trail Saving
* Trail Sharing
* Trail Riding
* Trail Statistics
