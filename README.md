# Bike Trails 

<img src="https://github.com/finleyp/BikeTrail467/wiki/media/bike-icon.png" width="150" height="150">

A cross platform mobile application built using Flutter to allow users the ability to record bike trails.

Features:

* Trail Recording
* Trail Saving
* Trail Sharing
* Trail Riding
* Trail Statistics
